MASOCORE HOSTGATOR UPLOAD

1. Open HostGator cPanel and choose File Manager.
2. Open the public_html folder for masocore.media.
3. Upload the contents of this package: index.html, styles.css, and assets.
4. Extract the ZIP if you uploaded it as one archive.
5. Make sure index.html sits directly inside public_html, not inside another masocore-site folder.

No database, Node.js process, build command, or package installation is required.
